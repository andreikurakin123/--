# 🏥 Hospital Bot

Telegram-бот для управления записями пациентов в медицинском учреждении. Бот предоставляет удобный интерфейс для пациентов для записи на прием и для администраторов для управления записями.

## 🚀 Возможности

### Для пациентов:
- 📅 Запись на прием к врачу
- 👁️ Просмотр своих записей
- ✏️ Редактирование существующих записей
- ❌ Отмена записей
- 📋 Просмотр доступных специалистов

### Для администраторов:
- 👥 Управление всеми записями
- 📊 Статистика и отчеты
- 🔔 Уведомления о новых записях

## 🛠️ Технологии

- **Python 3.8+**
- **aiogram 3.x** - асинхронный фреймворк для Telegram Bot API
- **SQLite/PostgreSQL** - база данных
- **FSM (Finite State Machine)** - управление состояниями диалогов
